A優勢文字列 [Python]

問題文

`A` と `B` だけからなる文字列 \(S\) が与えられます。

この文字列が以下の性質を持つ場合、「A優勢文字列」と呼ぶことにします。

  • 左から何文字を取り出したとしても、必ずその中に含まれる `A` の文字数が `B` の文字数より少なくなることがない。

与えられた文字列が「A優勢文字列」かどうか、判定してください。

制約

  • \(1 ≦ N ≦ 100\)
  • \(S\) は `A` と `B` だけからなる

入力

\(N\) \(S\)
  • \(1\) 行目で文字列の長さ \(N\) が与えられます。
  • \(2\) 行目で文字列 \(S\) が与えられます。

出力

\(S\) が「A優勢文字列」であれば `Yes` を、そうでなければ `No` を出力してください。

入力例1

5
ABABA

出力例1

Yes

\(S\) を左から何文字か取り出した場合…

  • 1文字取り出すと `A` → `A` が \(1\) 文字、`B` が \(0\) 文字
  • 2文字取り出すと `AB` → `A` が \(1\) 文字、`B` が \(1\) 文字
  • 3文字取り出すと `ABA` → `A` が \(2\) 文字、`B` が \(1\) 文字
  • 4文字取り出すと `ABAB` → `A` が \(2\) 文字、`B` が \(2\) 文字
  • 5文字取り出すと `ABABA` → `A` が \(3\) 文字、`B` が \(2\) 文字

となり、「その中に含まれる `A` の文字数が `B` の文字数より少なくなることがない」ので、これは「A優勢文字列」である。よって `Yes` を出力する。

入力例2

4
ABBA

出力例2

No

左から \(3\) 文字取り出すと `ABB` となり、`A` が \(1\) 文字 `B` が \(2\) 文字である。

「その中に含まれる `A` の文字数が `B` の文字数より少なく」なってしまう場合があるので、これは「A優勢文字列」ではない。よって `No` を出力する。

入力例3

10
AAAAAAAAAA

出力例3

Yes
