大根 [Ruby]

問題文

武揚くんが、一本税込み X 円の大根を Y 本買う。必要な金額を求めよ。

制約

  • 0 ≤ X ≤ 10000
  • 0 ≤ Y ≤ 10000
  • X, Y は整数である。

入力

入力は以下の形式で標準入力から与えられる。

X Y

出力

大根を買うのに必要な金額を 1 行で出力せよ。

入力例1

200
5

出力例1

1000

一本 200 円の大根を 5 本買うので、200×5 = 1000 円 必要です。

入力例2

10000
10000

出力例2

100000000

入力例3

0
0

出力例3

0
4049
8124
32894076
257
29
