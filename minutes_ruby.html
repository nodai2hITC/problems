分 [Ruby]

問題文

X 分は何時間何分か、答えてください。

（「分」の部分は、0 以上 59 以下になるようにしてください。）

制約

  • 0 ≤ X ≤ 10000
  • X は整数である。

入力

入力は以下の形式で標準入力から与えられる。

X

出力

1 行目に時間、2 行目に分を出力してください。

入力例1

123

出力例1

2
3

123 分 = 2 時間 3 分なので、1 行目に 2 、2 行目に 3 を出力します

入力例2

0

出力例2

0
0

入力例3

10000

出力例3

166
40
6272
104
32
3132
52
12
1767
29
27
